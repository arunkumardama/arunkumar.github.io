What I Work On

I manage Identity and Access Management (IAM) operations for global enterprise users, building secure and scalable identity systems across multiple regions. My work focuses on maintaining strong access governance while ensuring seamless user access.

I specialize in identity lifecycle management (Joiner, Mover, Leaver), designing RBAC models, and integrating enterprise applications using SAML, OAuth 2.0, and OpenID Connect (OIDC) for secure Single Sign-On experiences.

I work extensively with Azure Entra ID and Microsoft 365, managing application identities, service principals, Exchange Online configurations, and enforcing access controls aligned with security best practices.

I also build PowerShell automation to streamline IAM processes, improve audit readiness, and reduce operational overhead.

From a security standpoint, I implement Conditional Access policies, monitor identity risks using Microsoft Defender, and perform access reviews to ensure compliance and protect enterprise environments.

Additionally, I manage secrets, certificates, and keys using Azure Key Vault to ensure secure integrations across enterprise systems.
